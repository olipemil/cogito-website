COGITO Workflow

Workflow Overview

The COGITO workflow involves a series of steps that start from the VASP calculations and proceed through the tight-binding model construction to the plotting of band structures and densities of states (DOS).

1. Static VASP Calculation

Initial calculations are performed using VASP with an indelible uniform grid. This provides the necessary data for generating the tight-binding model.

2. COGITO Main

The main COGITO code generates the basis and constructs the tight-binding model using the input from the VASP calculations.

3. COGITO TB

COGITO TB module reads all inputs and constructs the tight-binding model. This includes calculating overlaps and plotting the crystal bonds for comparison with the DFT calculations.

4. Band Structure & DOS

Using the TB model, the band structure and density of states are calculated. The plots can be projected onto the crystal bonds or displayed as a standard band structure plot.

5. Post-processing & Visualization

COGITO allows for plotting and post-processing of the results, including projecting the band structure or calculating the projected ICO band structure. This step enables the final analysis and interpretation of the results.
